# SweatWorks Frontend

## Prerequisites
- NodeJS (+20 version recommended)
- `npm` package manager.
- `sweatworks-backend` to be running.
- .env variables that should be in `.env` file:
```env
VITE_API_URL=http://localhost:3000 
```
## Install Deps
```bash
$ npm i
```

## Start Frontend
```bash
npm run dev
```
